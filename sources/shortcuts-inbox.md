# Shortcuts-Inbox

Temporäre Sammelliste für Shortcuts, die der `shortcut-capture`-Skill
automatisch befüllt, sobald im Chat ein Shortcut nachgeschlagen wird.

Einträge hier sind **noch nicht** Teil der veröffentlichten Dokumentation —
das übernimmt der `shortcut-curate`-Skill erst nach expliziter Aufforderung
und nach Rückfrage, welche Einträge tatsächlich in
`site/docs-shortcuts/` übernommen werden.

Format pro Eintrag:

```
- **<Shortcut>** — <Tool/OS>: <kurze Beschreibung, was er tut>. Vermutete Kategorie: `<slug>` (existiert bereits) | _neu? z. B. "<Vorschlag>"_. _Erfasst: <YYYY-MM-DD>_
```

## Offene Einträge

- **Esc** — Windows `cmd.exe`: löscht die aktuelle Eingabezeile (kompletten getippten Text vor dem Ausführen). Vermutete Kategorie: _neu? z. B. "Windows-Commandline"_. _Erfasst: 2026-07-01_
- **cls** — Windows `cmd.exe`: löscht den gesamten sichtbaren Konsolenbildschirm und setzt den Cursor neu (Pendant zu `clear` unter Linux). Vermutete Kategorie: _neu? z. B. "Windows-Commandline"_. _Erfasst: 2026-07-01_
